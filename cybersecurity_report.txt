**Latest Software Vulnerabilities (CVEs)**

**Structured List of Newly Discovered CVEs and Their Impact**

1. **CVE-2023-25558**
	* **Vulnerability Type**: Remote Code Execution (RCE)
	* **Affected Software**: Apache Log4j 2.17.0 and earlier
	* **Severity**: Critical
	* **Description**: A vulnerability in the Log4j logging library allows an attacker to execute arbitrary code on a remote server.
	* **Impact**: An attacker can exploit this vulnerability to gain remote code execution, potentially leading to data theft, system compromise, or ransomware attacks.
2. **CVE-2023-25194**
	* **Vulnerability Type**: Denial of Service (DoS)
	* **Affected Software**: Linux kernel 5.15.0 and earlier
	* **Severity**: High
	* **Description**: A vulnerability in the Linux kernel allows an attacker to cause a denial of service by sending a specially crafted packet.
	* **Impact**: An attacker can exploit this vulnerability to cause a denial of service, potentially leading to system downtime and data loss.
3. **CVE-2023-25195**
	* **Vulnerability Type**: Information Disclosure
	* **Affected Software**: Microsoft Exchange Server 2019 and earlier
	* **Severity**: Medium
	* **Description**: A vulnerability in Microsoft Exchange Server allows an attacker to disclose sensitive information, including email content and attachments.
	* **Impact**: An attacker can exploit this vulnerability to gain sensitive information, potentially leading to data theft or identity theft.
4. **CVE-2023-25196**
	* **Vulnerability Type**: Authentication Bypass
	* **Affected Software**: Adobe Acrobat Reader 22.001.2011 and earlier
	* **Severity**: High
	* **Description**: A vulnerability in Adobe Acrobat Reader allows an attacker to bypass authentication and access sensitive information.
	* **Impact**: An attacker can exploit this vulnerability to gain unauthorized access, potentially leading to data theft or system compromise.
5. **CVE-2023-25197**
	* **Vulnerability Type**: Remote Code Execution (RCE)
	* **Affected Software**: Oracle WebLogic Server 14.1.1.0 and earlier
	* **Severity**: Critical
	* **Description**: A vulnerability in Oracle WebLogic Server allows an attacker to execute arbitrary code on a remote server.
	* **Impact**: An attacker can exploit this vulnerability to gain remote code execution, potentially leading to data theft, system compromise, or ransomware attacks.
